# 2026 马年微信红包封面制作指南

## 微信红包封面规格

### 静态封面
| 参数 | 要求 |
|------|------|
| 尺寸 | 957 × 1278 px |
| 比例 | 3:4 |
| 格式 | PNG / JPG / JPEG |
| 大小 | ≤ 500KB |
| 色彩 | RGB |

### 动态封面（视频）
| 参数 | 要求 |
|------|------|
| 格式 | MP4 |
| 宽高比 | 3:4 |
| 分辨率 | ≤ 4K（推荐 1080P）|
| 时长 | 1 ~ 3 秒 |
| 帧率 | ≤ 30fps（推荐 25fps）|
| 码率 | ≤ 3000 Kbps（推荐 2800）|
| 编码 | H.264 |
| 大小 | ≤ 20MB |

### 封面故事视频
| 参数 | 要求 |
|------|------|
| 尺寸 | 1080 × 1800（3:5）|
| 码率 | ≤ 1600 Kbps |
| 其他 | 同动态封面 |

### 挂件（可选）
| 参数 | 要求 |
|------|------|
| 格式 | 透明 PNG |
| 用途 | 封面挂件 / 气泡挂件 |

## 制作流程

### 方式一：AI 全自动流程（推荐）

```
即梦AI生图（3:4, 1020×1360）
       ↓
即梦AI图生视频（5秒动画）
       ↓
剪映调整参数导出（1080P, 2800Kbps, H.264, 25fps, 1-3秒）
       ↓
上传到微信红包封面平台
```

### 方式二：微信内置个人封面（无需100粉丝）

```
AI生成静态图片（3:4比例）
       ↓
微信 → 发红包 → 红包封面 → 制作我的封面 → 从相册选择
```

### 方式三：官方平台定制（需100粉丝）

```
AI生成图片/视频
       ↓
cover.weixin.qq.com 登录
       ↓
定制封面 → 上传素材 → 提交审核（3-5天）
       ↓
付费购买（¥1/个，最少100个）
       ↓
生成领取链接/二维码分发
```

## 设计趋势（2026）

### 主流风格
1. **国潮风** — 传统元素+现代插画，最受欢迎
2. **3D立体** — C4D渲染，金属质感，高级感强
3. **水墨画** — 传统与现代融合，文化底蕴
4. **萌系卡通** — 可爱小马，年轻人最爱
5. **极简现代** — 干净构图，高级配色

### 动态趋势
- 金属光泽渐变（模拟金属反光）
- 粒子特效（金色粒子喷射）
- 马鬃飘动（流体动力学模拟）
- 互动挂件（手指触发动画）

## 审核注意事项

### 必须
- 原创设计或拥有版权
- 提供设计过程截图（AI生成记录即可）
- 内容与视频号创作方向相关

### 禁止
- 第三方品牌 logo、明星肖像
- 直接售卖或转让红包封面
- 违反公序良俗的内容
- 涉及宗教、烟草、医疗等敏感领域
